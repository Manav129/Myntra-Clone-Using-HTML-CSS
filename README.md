# Myntra Clone

A responsive, static clone of the Myntra landing page built with plain **HTML** and **CSS**.

**Live demo:** https://Manav129.github.io/Myntra-Clone-Using-HTML-CSS/

---

## Features
- Responsive header with search and action icons
- Banner and product category sections
- Static product cards and offers
- Easy to extend to React/Firebase or add a backend

## Files
- `index.html` — main page
- `index.css` — styles
- `Images/` — assets (logo, banners, categories, offers)

## How to view (locally)
1. Clone the repo:
   ```bash
   git clone https://github.com/Manav129/Myntra-Clone-Using-HTML-CSS.git
   ```
2. Open `index.html` in your browser.

## Notes
- Images and asset paths are case-sensitive on GitHub Pages — keep file names unchanged.
- To publish updates, commit and push to the `main` branch; the GitHub Pages site updates automatically.
